# Requirements

1. Python 3.X with pip/Anaconda on Windows
2. Java 1.8 or higher

# Install

Alternatively:
```
tuo/ml $ poetry install 
tuo/ml $ poetry 
```

Alternatively:
```
tuo/ml $ pip install .
```

If you do not have administrator rights append --user to above command.


# Run it

```
python train.py --help
python train.py
```
